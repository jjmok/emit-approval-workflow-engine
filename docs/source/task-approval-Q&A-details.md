## What counts as "working time" when calculating deadlines and escalations?

This is only for the KPI’s

## Should a request's details be editable after submission? 
Yes, its locked, only can add notes, if clarifications are required by an approver, any changes would require re-approval, start from zero

## Should requests be cancellable before approval? 
Yes, at any stage, it can be cancelled

## If someone raises a task, can they also be assigned as an approver?
nope, any process should always be an approver to the next step
